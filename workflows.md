# Starter Workflows

1. Start in demo mode.
2. Run the signed local smoke tests.
3. Replace `MarketplaceClient.php` with real marketplace calls.
4. Configure real credentials through environment variables.
5. Optionally run `tools/mock_marketplace.php` for adapter development without a real marketplace.
6. Run `php tools/run_conformance.php http://127.0.0.1:18080/api.php` before connecting Unicorn 2.
